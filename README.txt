TabTopics Course Format

A moodle course format that uses jquery tabs to display each section.

Prerequisites: None
Installation: Copy the folder into moodle/course/format and name the folder "tabtopics"
To change the default color (red) for the tabs, you can place the following CSS
code in your theme file:

.ui-tabs-selected { background-color: <NEW_COLOR> !important; background-image: none !important; }

Refer to the PDF documentation for more information: URL http://oohoo.biz/
